# lightDom

## 使用方法

```
<div class='box' id="div"></div>

$$('#div')
$$('.box')
$$('div')
```

## 支持

dom 查找

-   $$('#id') $$('.class') $$('tag name')
-   parent()
-   children()

dom 修改

-   addClass()
-   removeClass()

dom 删除

-   remove()

dom 增加

-   append()

dom 属性

-   html()
-   attr()
-   css()

事件绑定

-   click()
-   bind()
-   unbind()

显示隐藏

-   show()
-   hide()
